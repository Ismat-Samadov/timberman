# Selling Short Publisher as Software (Non-SaaS)

## What You're Selling

A self-hosted Next.js dashboard + Python pipeline that automates short-form video generation and publishing to YouTube. Fully open architecture — buyer owns and runs everything.

---

## Marketplaces

### Tier 1 — Highest Potential

| Platform | Why it fits | Format |
|---|---|---|
| **Gumroad** | #1 for code/templates, huge developer audience, easy setup | ZIP + docs |
| **Lemon Squeezy** | Cleaner than Gumroad, better for devs, handles EU VAT automatically | ZIP + docs |
| **Whop** | Fastest growing for automation/AI tools, optional community upsell | Access + Discord |

## Positioning Angles

### "Developer Tool / Boilerplate"
- Target: technical creators and developers who want full control
- Price range: $49–$199 one-time
- Best platforms: Gumroad, Lemon Squeezy, CodeCanyon

### "AI Content Machine / Automation System"
- Target: YouTubers, marketers, agencies automating Shorts/Reels
- Price range: $97–$497 one-time (higher perceived value)
- Best platforms: Whop, AppSumo, Gumroad

---

## Blockers to Solve Before Listing

1. **Setup complexity** — requires NeonDB, Cloudflare R2, GitHub Actions, ElevenLabs, fal.ai, YouTube OAuth. Need a polished setup guide or video walkthrough, otherwise refund requests pile up.

2. **API cost transparency** — each video costs ~$3–5 in API fees (Kling 2.6 Pro is the main driver at ~$1.40/clip). Buyers must know this upfront.

3. **GitHub Actions limits** — free tier is 2,000 min/month (~40 videos/month). Power users will hit this. Document it clearly.

4. **No one-click install** — this is a technical-user product. Price and market accordingly. Avoid targeting non-developers without a managed setup add-on.

---

## Recommended Starting Path

1. Package as a ZIP with clean docs
2. List on **Gumroad** at $97–$147
3. Launch on **ProductHunt** for initial traffic spike
4. If traction builds → approach **AppSumo** for a lifetime deal campaign
