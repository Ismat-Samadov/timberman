## Product Description (full page body)

```
Turn a topic into a published YouTube Short in one click.

Short Publisher is a self-hosted dashboard + AI pipeline that handles everything automatically:

1. Script — Claude AI writes a viral, hook-first script with visual prompts per segment
2. Voice — ElevenLabs generates studio-quality narration with word-level timestamps
3. Video — Kling 2.6 Pro generates cinematic AI video clips in parallel
4. Captions — TikTok-style animated captions (white/yellow, uppercase) burned in with FFmpeg
5. Publish — uploads directly to your YouTube channel via YouTube Data API

───────────────────────────────

WHAT YOU GET

→ Next.js admin dashboard (topics queue, video history, pipeline logs, settings)
→ Python pipeline scripts (fully documented, 8-step orchestrator)
→ GitHub Actions workflow (free compute — no server needed)
→ NeonDB schema + Drizzle migrations (free tier is enough)
→ Complete setup guide + per-service documentation

───────────────────────────────

REQUIREMENTS

You need accounts (all have free tiers or pay-per-use):
• Vercel — free hosting
• NeonDB — free Postgres database
• GitHub — free Actions compute
• Anthropic — Claude API (~$0.30/video)
• ElevenLabs — voice API (~$0.10/video)
• fal.ai — Kling 2.6 Pro video clips (~$4.20/video)
• YouTube Data API — free (via Google Cloud Console)

───────────────────────────────

RUNNING COSTS

This is a one-time purchase. API costs are separate:
• ~$4.60 per video total
• ~$138/month at 1 video/day
• ~$46/month at 10 videos/month

You pay providers directly — no middleman markup.

───────────────────────────────

WHO IS THIS FOR

✔ Developers and technical creators who want full control
✔ YouTubers building a Shorts presence without manual editing
✔ Agencies automating content for multiple niches
✔ Anyone who wants to own their AI video stack instead of paying $50–200/month for SaaS

NOT for complete beginners — basic familiarity with Vercel, GitHub, and terminal is needed.
```
